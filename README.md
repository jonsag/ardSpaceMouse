# ardSpaceMouse

An attempt to make a mouse

## Part list

* 1 x Arduino Micro
* 1 x Joystick module, [https://www.switchelectronics.co.uk/dual-axis-joystick-module](https://www.switchelectronics.co.uk/dual-axis-joystick-module)
* 6 x M2.5 screw
* 1.5mm screw
* 3 x Micro switch
* M2.5 threaded insert
